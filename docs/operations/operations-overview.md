# Operations Overview

- Runtime health: actuator + module-specific health/status APIs.
- Scheduling: multiple fixed-delay schedulers for reminders, AI jobs, AI calls dispatch/reconciliation.
- Provider operations: readiness checks and test-send APIs.
- Incident focus areas: provider outages, queue buildup, webhook mismatch, tenant misconfiguration.
